1. Executive Summary & Kinematics

This report provides a formal evaluation of the 2-DOF parallel ankle joint mechanism (RSS/U-Joint topology) to guide the selection of servo actuators. Standard parallel mounting is compared side-by-side with a decoupled orthogonal yaw-tilted configuration. These configurations are analyzed across two ankle spread values (E = 2.3 cm and E = 5.0 cm) using verified dimensions matching the physical robot drawing.

1.1 Mechanical Dimensions Reference

The calculations are based on the verified layout dimensions: A (Crank r₁) = 2.5 cm, B (Crank r₂) = 2.5 cm, C (Rod L₁) = 24.3 cm, D (UJ to M2 relative height) = 16.0 cm, E (Ankle spread) = 2.3 cm / 5.0 cm, G (Foot to UJ Height) = 3.6 cm, H (UJ to M1 relative height) = 23.5 cm, I (Rod L₂) = 16.3 cm, L (Y-Offset/depth) = 3.8 cm.

1.2 Kinematics & Jacobians

The ankle joint is constrained by a central Universal Joint that permits Pitch (α) and Roll (β) rotations while blocking Yaw and translation. The pushrod lengths are constrained by physical joint distance equations. Differentiating these constraints gives the system Jacobian J, which maps ankle velocities to motor speeds: θ̇ = J · ẋ.

By the Principle of Virtual Work, the torque transmission is mapped as: τ_ankle = Jᵀ · τ_motor

This mapping allows us to sweep the complete workspace (Pitch from -25° to +25° and Roll from -40° to +40°) to find the minimum guaranteed torque output for any position.

6. Comparison & Design Recommendations

The key findings and engineering recommendations from this sizing analysis are:

  • Orthogonal Layout Performance: Tilted mounting decoupling improves roll torque capability by over 2.5x in narrow configurations (112.6 N·m vs 44.4 N·m peak on RS03). It is the recommended design configuration for roll stability.
  • Effect of Spread (E): Widening the ankle spread to 5.0 cm improves standard layout roll performance considerably (to 119.0 N·m). If mechanical layout permits, the Orthogonal configuration with a 5.0 cm base provides the absolute best, most isotropic performance (150 N·m Pitch / 145 N·m Roll).
  • Motor Sizing Verdict: The Robstride RS03 is the optimal actuator selection. It guarantees sufficient continuous rated control torque and offers peak torque capacities up to 112 N·m for ankle recovery and landing impact stabilization.
